- What is the MongoDB Database?
	- NoSQL - no rows and columns
	- Data stored in documents
		- Document - way to organize and store data (field-value pairs)
	- Documents stored in collections of documents
	- Database - collection of collections
- MongoDB Atlas - cloud database
	- Clusters - groups of servers for storing data
	- Replica set - collection of instances
	- Clusters configured as replica sets
